Angular directive

view-object source

Description

Angular directive for including HTML object tag dynamically.

Attributes

NameTypeDescription
data String

Required. Url of file to be viewed. Url must be trusted upfront.
type String

Optional. Mime-type of file to be viewed.
height String

Optional. Height value to be applied to object tag.
width String

Optional. Width value to be applied to object tag.

Example

<view-object data="ctrl.viewUri" type="ctrl.contentType" height="'600px'" width="'100%'"></view-object>
