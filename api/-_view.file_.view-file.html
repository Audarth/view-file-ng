Angular directive

view-file source template

Description

Angular directive for viewing files. Leverages a.o. highlightjs, json-explorer, sanitize, videogular, x2js.

Attributes

NameTypeDescription
data String

Optional. Data of file to be viewed. Do not use together with uri.
uri String

Optional. Url of file to be viewed.
content-type String

Required. Mime-type of file to be viewed.
allow-modal Boolean

Optional. Allow opening of file in modal overlay. Default: true.
controls Boolean

Optional. Show controls on left. Default: true if download-uri or allow-modal.
download-uri String

Optional. Url of file for download purpose. Default: null.
file-name String

Optional. Filename for display. Default: uri portion after last /.
show-code Boolean

Optional. Show raw code initially for JSON, HTML, Text, and XML. Default: false.
template String

Optional. Url of HTML template to use for this directive.
trust-uri Boolean

Optional. Apply trustAsResourceUrl on uri (not recommended). Default: false.

Example

<view-file uri="ctrl.viewUri" content-type="ctrl.contentType"
  allow-modal="true" controls="true" download-uri="ctrl.downloadUri" file-name="ctrl.fileName"
  show-code="false" template="/my/view-template.html" trust-uri="false">
</view-file>
or
<view-file data="ctrl.viewData" content-type="ctrl.contentType"
  allow-modal="true" controls="true" download-uri="ctrl.downloadUri" file-name="ctrl.fileName"
  show-code="false" template="/my/view-template.html" trust-uri="false">
</view-file>
